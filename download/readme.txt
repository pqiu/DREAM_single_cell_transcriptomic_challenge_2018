To run the matlab scripts, this folder should contain the raw data files provided in the DREAM_single_cell_transcriptomic_challenge.

Files should include: 
- bdtnp.txt
- binarized_bdtnp.csv
- dge_binarized_distMap.csv
- dge_normalized.txt
- dge_raw.txt
- DREAM_SCTS_TeamX_20genes.csv
- DREAM_SCTS_TeamX_40genes.csv
- DREAM_SCTS_TeamX_60genes.csv
- geometry.txt

All these files are available througth the challenge (https://www.synapse.org/#!Synapse:syn15665609/wiki/582909)
